# pi-web-ui

An agent-driven web interface built via [pi-web-architect](https://github.com/LPC3333/pi-web-architect).

## Quick Start

Click `start.bat`. It launches both the server and client.

## Structure

client/   — React + TypeScript frontend (Vite)
server/   — Node.js + TypeScript backend
shared/   — Shared types

## Tech

TypeScript, React, Vite, Node.js
